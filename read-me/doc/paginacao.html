Paginação

Todas as configurações básicas da paginação são encontradas no arquivo de configuração. Abaixo você vê os tipos disponíveis. Como as configurações além do tipo são alinhamento e tamanho de fonte, não demonstro essas opções abaixo.

A paginação é gerada através de um modelo padrão do Ruby chamado através do seguinte código no ruby:

HTML

<%= paginate @view %>

Não precisa colocar a paginação dentro de <div> nem nada.

É necessário editar o arquivo config/locales/pt-BR.yml acrescentando o seguinte trecho:

pt-BR.yml

views:

pagination:

first: "<span title='Primeira Página'><svg class="ico--esquerda"><use xlink:href="/assets/ico.svg#seta-dupla" /></svg></span>"
last: "<span title='Última Página'><svg class="ico"><use xlink:href="/assets/ico.svg#seta-dupla" /></svg></span>"
previous: "<span title='Anterior'><svg class="ico--esquerda"><use xlink:href="/assets/ico.svg#seta-fina" /></svg></span>"
next: "<span title='Próxima'><svg class="ico"><use xlink:href="/assets/ico.svg#seta-fina" /></svg></span>"
truncate: "..."

Por conta da forma como trabalha, você não precisa acrescentar mais nada e nem trabalhar no CSS, somente no arquivo de configuração.

A paginação é formada por angular e o HTML gerado é similar a esse:

HTML

<ul class="pagination">

<li>

<a href="#" aria-label="Previous">

<span aria-hidden="true">«</span>

</a>

</li>

<li><a href="#">1</a></li>

<li class="active"><a href="#">2</a></li>

<li><a href="#">3</a></li>

<li class="disabled"><a href="#">4</a></li>

<li><a href="#">5</a></li>

<li>

<a href="#" aria-label="Next">

<span aria-hidden="true">»</span>

</a>

</li>

</ul>

Toda a formatação é feita através do arquivo de configuração e do arquivo de cores.
