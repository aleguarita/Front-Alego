Formulários com erro

Nossos sistemas estão fazendo uso de angular para indicar que há um erro no formulário. Abaixo como utilizar (até o momento existe apenas um tipo de configuração, sem variação).

O elemento do formulário (exemplo: .form-texto) deve receber a classe extra .form--erro quando ela apresentar erro (configurar o angular para isso).

Depois do elemento do formulário incluir um <div> com a classe .form--erro__bloco. Dentro dele colocar cada mensagem com a indicação adequada do angular para o tipo de erro (pode ser em <div> ou <p>, mas o <div> é mais adequado por não apresentar margens).
