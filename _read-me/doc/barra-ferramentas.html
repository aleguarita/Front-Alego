Barra de Ferramentas

As configurações básicas da abas são encontradas no arquivo de configuração. Abaixo você vê os tipos disponíveis e suas variações.

Cria uma barra de ferramentas para colocar botões e itens de formulário.

A aparência dela muda ligeiramente conforme o tipo de formulário escolhido. Veja as opções nas configurações.

As classes usadas são

.barra-ferramentas
A classe básica. Já formata como flex e dispensa o uso de .form-container (os formulários podem usar .field-x normalmente)
.barra-ferramentas__separador
Distancia e separa o elemento atual dos demais
.barra-ferramentas__acoes
Para alinhar o botão à direita, no final

O código dos exemplos a seguir é:

HTML

<section class="barra-ferramentas">

<div class="field-3 form-texto">

<input type="text" placeholder="Pesquisa exemplo" />

<button class="form__ico"><svg class="ico"><use xlink:href="../images/ico.svg#lupa" /></svg></button>

</div>

<div class="form-icones barra-ferramentas__separador">

<label for="modo-tabela"><svg class="ico"><use xlink:href="../images/ico.svg#lista" /></svg></label>

<label for="modo-card"><svg class="ico"><use xlink:href="../images/ico.svg#card" /></svg></label>

</div>

<div class="form-select">

<select id="sort">

<option>Exemplo</option>

<option>Exemplo</option>

</select>

</div>

<div class="barra-ferramentas__acoes">

<button class="bt">Botão</button>

<button class="bt">Botão</button>

</div>

</section>
