Loading

O loading é um <div> que você deve inserir dentro da sua página. Não importa onde você vai colocar, ele vai ocupar a tela inteira (mas o ideal é que seja filho direto do <body>).

A imagem já está embutida na GEM. Caso você não esteja trabalhando em RAILS ou queira modificar a imagem, ela se encontra na pasta /images

O código de HTML a seguir é pensando em colocar a animação ao lado do texto "Carregando". Evidentemente você pode alterar o texto ou a disposição dos elementos (por usar flex um <br /> não vai fazer a imagem ficar acima do texto, neste caso faça alterações no CSS).

Se quiser fazer uso de outra imagem animada para isso, basta colocar na pasta /assets/images

HTML

<div class="loading">

<img src="<%= asset_path("loading.svg") %>" alt="Carregando">

Carregando

</div>

O resultado será parecido com isso (mas bloqueando a tela inteira):
