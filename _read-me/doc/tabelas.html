Tabelas

Todas as configurações básicas de tabela são encontradas no arquivo de configuração. Abaixo você vê todas as configurações.

Para chamar uma tabela você tem duas classes disponíveis (desde que ativas em $uso-classes, conforme descrito no arquivo de configuração):

.tab
<table>. A classe básica para todas as tabelas
.tab--form
<table>. Use para tabelas formulários (estilo planilha)

Caso precise de variantes ou de modificar a classe básica para atender à necessidade do layout, faça-o em um arquivo tabela.scss dentro da pasta extra/.

Existem algumas classes para serem usadas dentro das tabelas:

.tab__acoes
<td>. Use no <td> que contém os ícones de ações
.col--X
<th>. Use nas colunas de título (<th>). X é substituído pelo nome que você criou dentro do arquivo de config e determina a largura desta coluna. No momento já temos criada a .col--acoes

Regras para a tabela responsiva

No arquivo de config/+config.scss é possível determinar o tipo de media query usado para isso (o padrão é tamanho máximo, mas você pode decidir por orientação para sempre exibir "responsiva" quando a largura da tela for menor que a altura).

Para que a tabela responsiva funcione, é necessário inserir o atributo data-title em cada uma das células e dentro dela colocar o mesmo texto que aparece no <th> (incluindo acentos e espaço, pois este texto será escrito pelo CSS). O código da tabela seria como o seguinte:

HTML

<table class="tab">

<thead>

<tr>

<th>Nome do Usuário</th>

<th>Matrícula</th>

<th>Status</th>

</tr>

</thead>

<tbody>

<tr>

<td data-title="Nome do Usuário">...</td>

<td data-title="Matrícula">...</td>

<td data-title="Status">...</td>

</tr>

<tr>

<td data-title="Nome do Usuário">...</td>

<td data-title="Matrícula">...</td>

<td data-title="Status">...</td>

</tr>

<tr>

<td data-title="Nome do Usuário">...</td>

<td data-title="Matrícula">...</td>

<td data-title="Status">...</td>

</tr>

</tbody>

</table>
Não coloque item de ação em células diferentes, mesmo que você tenha configurado para aparecer como bloco
